## About this project

GrapheneOS is a privacy and security focused mobile operating system built on the Android Open Source Project. It hardens the base system with stronger sandboxing, exploit mitigations, and permission controls while keeping broad compatibility with mainstream Android apps. Users can install Play services through GrapheneOS's sandboxed compatibility layer instead of granting them privileged access inside the OS.

The project also ships privacy and security focused apps and services, including the hardened Vanadium browser and WebView, Auditor for device verification, a secure PDF Viewer, and a modern Camera app. GrapheneOS pairs deep technical work with a clean installation path, including a web installer that makes deployment much easier for supported devices.

## Why fund it?

GrapheneOS gives people a realistic way to use mobile devices with far stronger privacy and security guarantees. That matters for Bitcoin users, developers, journalists, and anyone operating in hostile environments, but the project's impact reaches further because parts of its work have been upstreamed into AOSP, Linux, and other core infrastructure.

OpenSats awarded GrapheneOS a $1M grant in early 2023. We continue to support bitcoin-adjacent privacy and security tools because people need them to use open networks in a secure and self-sovereign way.

## What's next?

GrapheneOS continues to improve the security model of Android without sacrificing usability. Current work includes hardening the OS against real-world attacks, maintaining strong app compatibility, improving first-party privacy and security tools, and making GrapheneOS easier to deploy on supported hardware.

You can explore GrapheneOS features at https://grapheneos.org/features, see supported devices at https://grapheneos.org/faq#device-support, and install it via the web installer at https://grapheneos.org/install/.
