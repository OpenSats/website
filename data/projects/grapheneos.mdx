## About this project

GrapheneOS is a mobile operating system built on the Android Open Source Project. It adds stronger sandboxing, exploit mitigations, and permission controls while keeping broad compatibility with Android apps. Users can install Play services through GrapheneOS's sandboxed compatibility layer instead of giving them privileged access inside the OS.

The project also ships its own privacy and security tools, including the Vanadium browser and WebView, Auditor for device verification, a secure PDF Viewer, and a Camera app. Supported devices can be installed through the project's web installer.

## Why fund it?

GrapheneOS gives people a phone OS with much stronger privacy and security properties than stock Android. That matters for Bitcoin users, developers, journalists, and anyone working in hostile environments. Parts of the project's work have also been upstreamed into AOSP, Linux, and related infrastructure.

[OpenSats awarded GrapheneOS a $1M grant in early 2023](/blog/2023-year-in-review#grantees). We support privacy and security tools that help people use open networks on their own terms.

## What's next?

Current work includes OS hardening, app compatibility, first-party privacy and security tools, and easier deployment on supported devices.

Recent GrapheneOS news points to broader device support ahead. The project announced work to support devices beyond Pixels, and [Motorola confirmed a partnership with the GrapheneOS Foundation for future official support](https://9to5google.com/2026/03/01/motorola-confirms-grapheneos-partnership-for-a-future-smartphone-porting-features/).

You can explore [GrapheneOS features](https://grapheneos.org/features), see [supported devices](https://grapheneos.org/faq#device-support), and install it via the [web installer](https://grapheneos.org/install/).
