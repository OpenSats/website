Bitcoin Core is the reference implementation of the Bitcoin protocol. It is the full node software that validates blocks and transactions, enforces consensus rules, relays data across the peer-to-peer network, and gives users a way to verify the system for themselves.

Core contributors review and test pull requests, fix bugs, improve performance, maintain wallet and RPC interfaces, and keep the codebase understandable enough for new contributors to join. Bitcoin depends on that work.

## Why fund it?

Bitcoin Core needs sustained review, testing, maintenance, and onboarding. That requires experienced contributors and enough support for newer developers to build context over time.

OpenSats has funded Bitcoin Core work since launching its [LTS program for contributors](/blog/announcing-lts-grant-program-to-support-bitcoin-core-contributors) in 2023. Since then, that support has expanded across long-term grants for maintainers, targeted grants for review and testing work, and adjacent efforts such as the [Bitcoin Core App](/blog/caring-for-bitcoin-core#bitcoin-core-app) and the [Bitcoin Core Testing Suite](/blog/bitcoin-grants-feb-2024#bitcoin-core-testing-suite). Later rounds added support for contributors working on areas like Initial Block Download, release testing, test coverage, build system maintainability, and peer-to-peer privacy research.

## What's next?

The backlog is still large. Bitcoin Core needs steady review, release testing, better testing infrastructure, and long-term work on areas like `libbitcoinkernel`, package relay, wallet improvements, sync performance, and peer-to-peer privacy. It also needs contributors who can build context, review hard changes, and take on maintenance over time.

If you want a broader picture of the work OpenSats is helping fund around the project, start with [Caring for Bitcoin Core](/blog/caring-for-bitcoin-core), [Additional Grants for Bitcoin Core Contributors](/blog/more-for-core), and [5 Grants to Strengthen Bitcoin Development](/blog/five-grants-to-strengthen-bitcoin-development).
