GrapheneOS is an Android-based operating system that replaces most of the Google components with open-source equivalents and hardens the rest of the platform against a long list of attacks. It runs on recent Pixel phones.

Notable features include a hardened memory allocator, per-app network and sensor permissions, an optional sandboxed Google Play layer, and verified boot with user-supplied signing keys. Bitcoin and Nostr users often pick GrapheneOS as a base because it lets them install wallets and clients without an attached Google account.

The project lives at [grapheneos.org](https://grapheneos.org/), with source at [github.com/GrapheneOS](https://github.com/GrapheneOS).
